# Генерация контента с контролем на этапе создания

Проект предназначен для генерации контента с встроенным контролем целостности на этапе создания. Система обеспечивает согласованность и логическую непротиворечивость генерируемых элементов.

## Архитектура системы

Система поддерживает генерацию контента различных модальностей с использованием специализированных моделей и условий целостности.

### Таблица модальностей и моделей

| Модальность | Модель | Условия целостности |
|-------------|--------|---------------------|
| Текст описания зоны | LLM (LLaMA/Mistral) | Непротиворечивость с предыдущими зонами, временная логика |
| 2D-текстуры/карты | Stable Diffusion + ControlNet | Наличие необходимых деталей (ручка двери, выключатель) |

## Компоненты системы

### 1. Генерация текста описания зоны

**Модель:** LLM (LLaMA/Mistral)

**Условия целостности:**
- **Непротиворечивость с предыдущими зонами** — проверка на соответствие ранее созданным описаниям зон
- **Временная логика** — обеспечение последовательности событий и временных рамок в описании

### 2. Генерация 2D-текстур и карт

**Модель:** Stable Diffusion + ControlNet

**Условия целостности:**
- **Наличие необходимых деталей** — автоматическая проверка на присутствие ключевых элементов:
  - Ручка двери
  - Выключатель
  - Другие обязательные элементы сцены

## Принцип работы

1. **Инициализация** — выбор модальности и настройка параметров генерации
2. **Генерация контента** — создание контента с помощью соответствующей модели
3. **Контроль целостности** — проверка сгенерированного контента на соответствие условиям целостности
4. **Валидация** — подтверждение или отклонение результата с возможностью повторной генерации

## Требования

- Python 3.8+
- Зависимости для работы с LLM (LLaMA/Mistral)
- Зависимости для работы со Stable Diffusion и ControlNet

## Использование

### Быстрый старт

```python
from src import TextZoneGenerator, TextureGenerator, IntegrityChecker

# Инициализация генераторов
text_gen = TextZoneGenerator()
texture_gen = TextureGenerator()
checker = IntegrityChecker(text_gen, texture_gen)

# Генерация текста с проверкой целостности
zone = text_gen.generate_zone(
    zone_id="zone_001",
    prompt="Тёмный коридор с деревянными стенами",
    timestamp="10:00"
)

# Проверка целостности текста
result = text_gen.check_integrity(zone)
print(f"Валидно: {result.is_valid}")
print(f"Оценка согласованности: {result.consistency_score}")

# Генерация текстуры с требуемыми деталями
texture = texture_gen.generate_texture(
    texture_id="door_001",
    prompt="Деревянная дверь в старом доме",
    required_details=["ручка двери", "замок"]
)

# Проверка наличия деталей
detail_result = texture_gen.check_details(texture)
print(f"Обнаруженные детали: {detail_result.detected_details}")
print(f"Отсутствующие детали: {detail_result.missing_details}")
```

### Генерация с автоматической валидацией

```python
from src import IntegrityChecker

checker = IntegrityChecker()

# Генерация текста с автоматической проверкой
content, result = checker.generate_with_validation(
    modality="text",
    zone_id="zone_002",
    prompt="Загадочная комната с древними артефактами",
    timestamp="10:15"
)

if result.is_valid:
    print("Генерация успешна!")
else:
    print(f"Проблемы: {result.issues}")
    print(f"Рекомендации: {result.suggestions}")

# Генерация текстуры с автоматической проверкой
texture, texture_result = checker.generate_with_validation(
    modality="texture",
    texture_id="room_001",
    prompt="Древняя комната с каменными стенами",
    required_details=["факел", "каменная текстура"]
)
```

## Графический интерфейс

Проект включает интерактивный графический интерфейс для удобной работы с системой генерации контента.

### Запуск визуализатора

```bash
python run_visualizer.py
```

### Возможности интерфейса

#### 📝 Вкладка "Генерация текста"
- Ввод ID зоны, временной метки и промпта
- Генерация описаний зон с помощью LLM
- Проверка целостности (согласованность и временная логика)
- Просмотр истории сгенерированных зон
- Отображение результатов проверки целостности

#### 🎨 Вкладка "Генерация текстур"
- Ввод ID текстуры и промпта
- Указание обязательных деталей (ручка двери, выключатель и т.д.)
- Генерация 2D-текстур через Stable Diffusion + ControlNet
- Проверка наличия требуемых деталей
- Загрузка и просмотр изображений

#### 📊 Вкладка "Обзор целостности"
- Общая статистика по зонам и текстурам
- Таблица всех зон с оценками согласованности и временной логики
- Таблица всех текстур с информацией о найденных/отсутствующих деталях
- Цветовая индикация статуса (✓ валидно / ✗ невалидно)

### Элементы управления

- **Меню "Файл"**: Сохранение/загрузка проекта, выход
- **Меню "Помощь"**: Информация о программе
- **Строка состояния**: Отображение текущих операций

### Скриншот интерфейса

Интерфейс предоставляет:
- Интуитивное управление генерацией
- Визуальную обратную связь о статусе целостности
- Возможность работы без знания программирования
